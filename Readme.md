# Личный проект «Седона»

#### Общая информация
Сайт состоит из двух основных страниц:

Главная страница с информацией о городе и его преимуществах

Страница каталога гостиниц с системой фильтрации и поиска

#### Основной функционал
Главная страница:
Приветственная секция с заголовком

Секция преимуществ города с фотографиями

Описание инфраструктуры (жилье, еда, сувениры)

Форма подписки на рассылку

Контактная информация и социальные сети

Страница каталога:
Система фильтрации гостиниц по:

Инфраструктуре (бассейн, парковка, Wi-Fi)

Типу жилья (гостиница, мотель, апартаменты)

Ценовому диапазону

Различные варианты отображения карточек

Пагинация

Форма подписки

#### Используемые технологии

HTML5 - семантическая разметка

CSS3 - стилизация, включая:

CSS Grid и Flexbox для построения layouts

CSS-анимации и transitions

SVG - иконки и векторная графика

#### Дополнительные особенности:
Доступность (ARIA-атрибуты, семантическая разметка)

Валидация форм

Модальные окна

Интерактивные элементы (tooltips, кастомные чекбоксы/радиокнопки)

Оптимизация изображений
